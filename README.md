## 프로젝트 설명

백엔드와 프론트를 연결하기 전 리액트의 기초를 알아보는 과정.

<br>

### ▫️ 배운 것

* useState를 통해 상태처리
* 비동기 작업을 처리할때 프로미스를 이용(callback, async&await 여기선 사용 x)
* fetch API를 활용해 JSON 본문을 읽어온다 (axios 라이브러리도 성격이 비슷하다)
* .env 파일을 만들어 환경변수로 api값 숨기기
